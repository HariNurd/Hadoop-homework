# WordCount dengan Hadoop Streaming di Google Colab

Proyek ini menampilkan proses word count menggunakan mapper dan reducer sederhana dengan Hadoop Streaming. Data yang digunakan adalah teks Pembukaan UUD 1945 (dari judul sampai paragraf terakhir) yang disimpan dalam file wordcount.txt. Seluruh proses dijalankan di Google Colab dengan standalone cluster Hadoop.

langkah pengerjaan tugas
- Mengambil teks Pembukaan UUD 1945 dan menyimpannya sebagai wordcount.txt.
- Membuat mapper sederhana (mapper.py) untuk menghasilkan pasangan kata dan frekuensi.
- Menjalankan Hadoop Streaming di Google Colab untuk memproses data tersebut.
- Menyimpan hasil output dalam file output_word_count.txt.
- Mengunggah hasil dan kode ke GitHub.
